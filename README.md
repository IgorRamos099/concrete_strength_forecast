
#  Previsão da Resistência do Concreto

Modelo preditivo para prever a resistência à compressão do concreto 
com base em variáveis como cimento, água, idade e outros componentes.
---

## Tecnologias utilizadas
- Python
- Pandas
- Scikit-learn (Random Forest + Regressão Linear)
- Matplotlib
---

## Resultados
- Random Forest: R² ≈ 0.82 | MAE ≈ 5.0 MPa
- Regressão Linear: R² ≈ 0.52 | MAE ≈ 9.3 MPa

---

## Dataset
Dados de composição de amostras de concreto com sua respectiva 
resistência à compressão medida.
